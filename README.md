# ebook
## 项目为ePub电子书用户提供“管理、阅读、标记、批注、分享、统计、索引”等功能
#### 1.服务端：NodeJs-express搭建，提供接口，返回单页面
#### 2.浏览器端：fetch异步通信与服务端数据交互
>##### 2.1 平台页面：ReactJs负责路由控制和页面渲染，AntD负责视觉样式
>##### 2.2 阅读器：HTML+CSS+jQuery负责文字渲染，Bootstrap负责视觉样式，epubjs负责解析epub电子书
